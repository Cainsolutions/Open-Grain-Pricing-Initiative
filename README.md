# Grain Break-Even & Pricing Tool

A free, non-commercial Excel tool designed to help grain growers
understand break-even prices and margin per tonne during harvest.

## What this tool does
- Calculates cost per tonne from paddock-level inputs
- Shows break-even price
- Displays margin per tonne at current market prices
- Includes simple bear / base / bull price scenarios
- Locked formulas to prevent accidental errors

## Who this is for
- Grain growers
- Farm managers
- Agronomy and advisory discussions
- Educational use

## How to use
1. Download the Excel file
2. Enter acres, tonnes, price, and variable costs
3. Review cost per tonne and margin per tonne
4. Use scenario analysis to assess downside and upside risk
